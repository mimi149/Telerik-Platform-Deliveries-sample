This is a sample app demonstrating the offline support in Telerik Backend Services JavaScript SDK.

What It Does

This app is a simplified version of an app used to handle delivery orders. The employees making the deliveries use it to set the status of a delivery order and enter comments. They can work without Internet connection and all changes they make are synchronized to the cloud whenever the device gets back online.

Testing Offline Storage

In this sample the switch between offline and online mode is bound to the connectivity status of the device. If you are using the AppBuilder simulator, you can simulate offline mode from the Network Connection tab. If you are testing this app on a real device, you have to switch off the Internet connection in order to go in offline mode. For your convenience the current mode is always displayed in the bottom of the app.
